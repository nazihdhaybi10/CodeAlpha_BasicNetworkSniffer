# CodeAlpha_BasicNetworkSniffer

## Project Overview
This project is a Basic Network Sniffer developed using Python and Scapy as part of the CodeAlpha Cyber Security Internship.

The program captures network packets in real time and displays useful information about network traffic.

## Features
- Capture live network packets
- Show source and destination IP addresses
- Detect TCP, UDP, ICMP protocols
- Show ports and packet size
- Save logs to a file
- Start and stop sniffer using menu

## Technologies Used
- Python 3
- Scapy
- Kali Linux
- VirtualBox

## Installation
pip install scapy

## Usage
sudo python3 sniffer.py

## Output Example
[time] source_ip:port -> destination_ip:port | protocol | size
